API Python Flask with Mongodb

Endpoint Método/Parâmetros Retorno
"/modality-students/[modalidade]/[data-inicial]/[data-final]"
  • Método: GET
  • Parâmetros:
    • Nome da modalidade
    • Data inicial no padrão [ano]-[mes]-[dia] Ex: 2019-01-11
    • Data final no padrão [ano]-[mes]-[dia] Ex: 2019-01-11
 Lista de estudantes filtrados por modalidade e período passado.
"/campus-courses/[campus]"
  • Método: GET
  • Parâmetros:
    • Nome do Campus
 Lista de cursos do campus passado.
"/count-campus-students/[campus]/[data-inicial]/[data-final]"
  • Método: GET
  • Parâmetros:
    • Campus
    • Data inicial no padrão [ano]-[mes]-[dia] Ex: 2019-01-11
    • Data final no padrão [ano]-[mes]-[dia] Ex: 2019-01-11
 Número de estudantes no campus dentro do período passado.
"/insert-students"
  • Método: POST
  • Parâmetros passados no corpo da requisição no seguinte formato:
    { "campus": "AQ", "curso": "TÉCNICO EM INFORMÁTICA", "data_inicio": "2016-05-20", "idade_ate_31_12_2016": "19.0", "modalidade": "EAD", "municipio": "Campo Grande", "nivel_do_curso": "SUBSEQUENTE", "nome": "lucas gregory", "ra": "8570.0" }
Caso inserido, retorna o estudante que foi passado
"/delete-students/[ra]/[campus]"
  • Método: DELETE
  • Parâmetros:
    • RA(ranking acadêmico)
    • Campus
 Caso o aluno exista e seja removido com sucesso, retornará a mensagem 'Documento excluido com sucesso', caso contrário a mensagem será 'Erro ao deletar documento'
